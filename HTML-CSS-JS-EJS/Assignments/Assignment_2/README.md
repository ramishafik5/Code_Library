This project is a personal portfolio website developed using Node.js, Express.js, and EJS templating. It features four main pages: Home, About Me, Projects, and Contact. The layout and responsiveness of the site were achieved using Bootstrap, while Font Awesome provided the icons used throughout the interface.

The project was created primarily by Rami Shafik, with only minimal external references. A few external tools and examples were used in accordance with course guidelines. Bootstrap 5 (https://getbootstrap.com/
) was implemented for layout structure, navbar responsiveness, and general component styling under the MIT License. Font Awesome (https://fontawesome.com/
) was used to include icons in the navigation bar and footer, licensed under CC BY 4.0. The initial Express app structure was generated using WebProject as a base and refined by referring to the official documentation and classroom examples. Additionally, the project uses the CDN resource https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css for header and navbar creation.

All remaining logic, routing, content, and design elements were independently written and implemented by the developer. The Bootstrap navbar was customized to display brand text, align links to the right, and include a responsive dropdown menu for smaller screens. The Express setup, including routes and static file handling, `was built from an understanding of Express.js developed in class. The use of EJS partials allowed for a modular and reusable structure across all pages, aligning with best practices taught in coursework.
